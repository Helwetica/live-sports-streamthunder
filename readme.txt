Tested up to: 4.4.2

Get automatically list of all kind of sports events for your site and put your own links.

== Description ==

With this plugin show automatically daily list of all kind of sports events on your site or just of sport you want. Put your own links on each sport event to send traffic wherever you want or show links the default link to brind your users usefull sites to watch every sport match.

== Installation ==

1. Upload the folder 'st-widget' found inside the zip to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Usage ==

1. Create widgets selecting the sports for each one on Live Sports Streamhunder tab in wp-admin.
2. Copy & Paste the generated shortcode and put it on a page, post or wherever you want to show the list of sport events.
3. Configure your widget so that it best fits your site on Display Options tab in wp-admin.

= Changing the CSS =

Display Options tab in wp-admin.
